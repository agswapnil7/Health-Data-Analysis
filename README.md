# Health Data Analysis: Hospital Efficiency & Insights

This repository contains an analysis of healthcare data, focusing on hospital operational efficiency and data preprocessing for health-related insights.

## 📊 Project Overview
The project analyzes hospital datasets to identify patterns in healthcare delivery and operational performance. It includes a cleaned version of the hospital records prepared for further statistical analysis or machine learning.

## 📂 File Descriptions
* **`healthcare.csv`**: The primary dataset containing healthcare records.
* **`Hospital_Data_Cleaned_Swapnil.csv`**: A processed version of the dataset where missing values and inconsistencies have been handled.
* **`health.ipynb`**: Jupyter Notebook containing the Exploratory Data Analysis (EDA) and data cleaning scripts.
* **`Healthcare_Analysis_Report.docx`**: A detailed report discussing the findings and metrics related to hospital performance.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Pandas, NumPy, Matplotlib
* **Focus:** Data Cleaning, Operational Analytics

## ⚙️ Key Workflow
1. **Data Sanitization:** Cleaning records to ensure accurate reporting on hospital metrics.
2. **Efficiency Analysis:** Evaluating hospital throughput and resource utilization based on the provided data.
3. **Visualization:** Creating charts to represent healthcare trends and distributions.
